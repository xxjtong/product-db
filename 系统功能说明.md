# 物联网产品中心数据库 — 系统功能说明

## 项目概述

物联网（IoT）产品中心数据库，面向物联网产品销售场景，支持产品结构化管理、解决方案组合、报价单生成等核心业务流程。

**技术栈**: FastAPI + SQLAlchemy 2.0 + SQLite(dev)/PostgreSQL(prod) | Vue 3 + Composition API + TypeScript + Vite

**本地环境**:
- 后端 API: http://localhost:8000 (文档: /docs)
- 前端: http://localhost:5173
- 账号: admin / admin

## 1. 后端 API（FastAPI）

### 产品模块 `/api/products` — 18个端点

| 端点 | 方法 | 功能 |
|------|------|------|
| `/products` | GET | 产品列表（分页 + 映射表JOIN筛选：通讯方式/协议/供电/品类/搜索） |
| `/products` | POST | 创建产品（同时写入映射表 + 图片） |
| `/products/{id}` | GET | 产品详情（含映射数据、图片、规格定义、依赖关系、变体） |

| `/products/compare` | GET | 多产品参数对比（返回对比矩阵） |
| `/products/export` | GET | 导出产品列表（Excel） |
| `/products/{id}/spec-sheet` | GET | 产品规格表（HTML） |
| `/products/upload-image` | POST | 上传图片（本地存储/S3） |
| `/products/download-image` | POST | 从URL下载图片到本地存储 |
| `/products/{id}/dependencies` | GET | 查询产品依赖关系 |
| `/products/{id}/dependencies` | POST | 添加依赖关系 |
| `/products/{id}/dependencies/{dep_id}` | PUT | 更新依赖关系 |

## 3. 数据模型

### 字典层
| 表 | 说明 | 数据量 |
|---|---|---|
| `manufacturer` | 厂商/品牌 | 1 (Milesight) |
| `dict_comm_method` | 通讯方式 | 17条 |
| `dict_comm_protocol` | 通讯协议 | 15条 |
| `dict_power_supply` | 供电方式 | 6条 |
| `dict_sensor_metric` | 传感指标 | 21条 |

### 核心层
| 表 | 说明 |
|---|---|
| `device_category` | 设备分类（2级树，含level字段） |
| `product` | 产品主表（精简：model/manufacturer_id，无数组列） |
| `product_image` | 产品图片（多图CDN URL，is_primary标记主图） |

### 映射层
| 表 | 说明 |
|---|---|
| `product_comm_method` | 产品-通讯方式（含details如频段） |
| `product_comm_protocol` | 产品-通讯协议（含direction：采集/转发/双向） |
| `product_power_supply` | 产品-供电（含voltage_range/battery_life） |
| `product_hardware_interface` | 产品-硬件接口（含interface_name/quantity/description） |
| `product_sensor_capability` | 产品-传感能力（含measure_range/accuracy/resolution） |

### 业务层
| 表 | 说明 |
|---|---|
| `suppliers` | 供应商（与manufacturer共存，supplier=代理商） |
| `solutions` / `solution_items` | 解决方案 + 明细 |
| `quotations` / `quotation_items` | 报价单 + 明细 |
| `bom_templates` / `solution_bom_snapshots` | BOM模板 + 方案BOM快照 |
| `category_spec_definitions` | 分类规格定义（动态spec表单） |
| `product_dependencies` | 产品依赖关系 |
| `users` | 用户表 |
