# 物联网产品中心数据库 — 系统功能说明

## 项目概述

物联网（IoT）产品中心数据库，面向物联网产品销售场景，支持产品结构化管理、解决方案组合、报价单生成等核心业务流程。

**技术栈**: FastAPI + SQLAlchemy 2.0 + SQLite/PgSQL | Vue 3 + TypeScript + Vite | Docker

**本地环境**:
- 后端 API: http://localhost:8000 (Swagger: /docs)
- 前端: http://localhost:5173
- 账号: admin / admin (DEV_MODE 自动创建)

---

## 1. 后端 API

### 产品模块 `/api/products` — 18 端点
完整的 CRUD + AI 识别 + 规格书 + 对比 + Excel 导入/导出

### 分类模块 `/api/categories` — 10 端点
2 级分类树 + 规格定义管理

### 方案模块 `/api/solutions` — 10 端点
BOM 管理 + 依赖完整性检查 + 缺失产品推荐

### 报价模块 `/api/quotations` — 8 端点
报价 CRUD + snapshot 模式 + xlsx 导出

### BOM 模板 `/api/bom-templates` — 6 端点
模板 CRUD + 快照保存/加载/xlsx 导出

### AI 模块 `/api/ai` — 4 端点
SSE 流式对话 + Tool Calling + 对话历史持久化 + Mock 回退

### 认证模块 `/api/auth` — 8 端点
JWT 登录 + 自注册 + 会话 + 字段可见性 + 管理员用户管理

### 管理模块 `/api/admin` — 8 端点
用户 CRUD + 登录日志 + 字段可见性 + AI Prompt + AI 用量 + 下载审计

### 字典模块 `/api/dicts` — 8 端点
通讯方式/协议/供电/传感指标/厂商

### 供应商模块 `/api/suppliers` — 4 端点
供应商 CRUD
