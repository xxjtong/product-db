# 产品数据库 — 系统功能说明

## 项目概述

物联网产品中心数据库，支持产品选型对比、规格书生成、方案设计、报价单生成。

**技术栈**: FastAPI + SQLAlchemy 2.0 + Pydantic | Vue 3 + TypeScript + Vite | Docker

**本地环境**:
- 后端 API: http://localhost:8000 (Swagger: /docs)
- 前端: http://localhost:5173
- 账号: admin / admin (DEV_MODE 自动创建)

## 3. AI 方案助手

### 工具能力
| 工具 | 功能 |
|------|------|
| `search_products` | 关键词/品类/通讯方式/协议/供电搜索，先精确后bigram回退 |
| `get_product_detail` | 完整产品详情 (通讯/协议/供电/接口/传感) |
| `list_categories` | 品类列表 |
| `create_quotation` | 自动添加产品到BOM → 创建报价单 |

### GenUI 动态组件
| 组件 | 触发条件 | 功能 |
|------|---------|------|
| `SolutionProductCard` | 搜索到产品 | 勾选 + 数量 + 加入方案 + 对比 |
| `QuoteDraftCard` | 创建报价单 | 绿色摘要卡片 + 查看按钮 |

### SSE 事件流
```
connect → tool → products → component(GenUI) → text → done → quick_replies
```

## 4. 方案 (Solution) 完整流程

```
① 客户信息 ─────────────────────────────────────
   客户名称 / 项目名称 / 状态 / 备注

② AI 方案助手 ──────────────────────────────────
   "10个温湿度传感器 + 1个LoRaWAN网关"
   → 🤖 AI推荐产品 (GenUI卡片)
   → ☑勾选产品 + 数量 → [加入方案(N)] [对比N]

③ 产品清单 (BOM) ────────────────────────────────
   [+ 批量选品] 弹窗搜索 + 多选 + 批量添加
   列表: 产品/数量/单价/折扣/小计/备注/删除

④ 依赖检查 ─────────────────────────────────────
   [依赖检查] → 缺件告警 + 推荐补齐
   依赖关系图 (Canvas可视化)

⑤ BOM 电子表格 ─────────────────────────────────
   [编辑BOM表格] → Univer编辑器 (列宽/合并/公式/样式)
   [保存] [导出xlsx] [另存为模板]

⑥ 生成报价单 ───────────────────────────────────
   [生成报价单] → 自动创建 → 跳转报价详情页
```
